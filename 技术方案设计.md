### 1.2 完整建表 SQL

```sql
-- ============================================
-- 字典表
-- ============================================

-- 省份
CREATE TABLE provinces (
    id          SMALLINT PRIMARY KEY,
    name        VARCHAR(20)  NOT NULL,
    region      VARCHAR(10)  NOT NULL,  -- 东北/华北/华东/华南/华中/西北/西南/海域
    sort_order  SMALLINT DEFAULT 0
);

-- 项目类型
CREATE TABLE project_types (
    id      SMALLINT PRIMARY KEY,
    code    VARCHAR(10)  NOT NULL UNIQUE,  -- onshore, offshore
    name    VARCHAR(20)  NOT NULL          -- 陆上风电, 海上风电
);

-- 项目状态
CREATE TABLE project_statuses (
    id         SMALLINT PRIMARY KEY,
    code       VARCHAR(20) NOT NULL UNIQUE,  -- approved,bidding,construction,grid_connected,completed
    name       VARCHAR(20) NOT NULL,         -- 核准,招标,在建,并网,完工
    sort_order SMALLINT DEFAULT 0            -- 生命周期顺序
);

-- 塔筒构造类型
CREATE TABLE tower_types (
    id      SMALLINT PRIMARY KEY,
    code    VARCHAR(10)  NOT NULL UNIQUE,  -- steel, hybrid, unknown
    name    VARCHAR(20)  NOT NULL          -- 钢塔, 混塔, 待确认
);

-- ============================================
-- 核心表
-- ============================================

-- 风电项目主表
CREATE TABLE projects (
    id               BIGSERIAL PRIMARY KEY,
    name             VARCHAR(200)  NOT NULL,            -- 项目名称
    province_id      SMALLINT      NOT NULL REFERENCES provinces(id),
    project_type_id  SMALLINT      NOT NULL REFERENCES project_types(id),
    capacity_mw      NUMERIC(8,2)  NOT NULL,            -- 总容量(MW)
    turbine_count    INT,                               -- 风机台数
    unit_capacity_mw NUMERIC(6,2),                      -- 单机容量(MW)
    tower_type_id    SMALLINT      DEFAULT 3 REFERENCES tower_types(id),  -- 默认: 待确认
    status_id        SMALLINT      NOT NULL REFERENCES project_statuses(id),
    owner            VARCHAR(200),                      -- 业主/开发商
    turbine_supplier VARCHAR(200),                      -- 风机供应商
    investment_bn    NUMERIC(10,2),                     -- 投资金额(亿元)
    approval_date    DATE,                              -- 核准日期
    bid_date         DATE,                              -- 招标日期（中标公示日）
    construction_start_date DATE,                       -- 开工日期
    completion_date  DATE,                              -- 完工/全容量并网日期
    planned_cod_date DATE,                              -- 计划并网日期
    description      TEXT,                              -- 项目描述/备注
    is_verified      BOOLEAN DEFAULT FALSE,             -- 是否人工核实
    created_at       TIMESTAMPTZ DEFAULT NOW(),
    updated_at       TIMESTAMPTZ DEFAULT NOW(),

    -- 唯一约束：同省同名项目视为同一项目
    CONSTRAINT uq_project UNIQUE (name, province_id)
);

-- 项目信息来源关联（多对多）
CREATE TABLE project_sources (
    id           BIGSERIAL PRIMARY KEY,
    project_id   BIGINT       NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
    source_id    SMALLINT     NOT NULL REFERENCES data_sources(id),
    source_url   VARCHAR(500),                     -- 原始链接
    captured_at  TIMESTAMPTZ DEFAULT NOW()
);

-- 项目状态变更日志（生命周期追踪）
CREATE TABLE project_status_logs (
    id          BIGSERIAL PRIMARY KEY,
    project_id  BIGINT    NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
    old_status_id SMALLINT,
    new_status_id SMALLINT NOT NULL REFERENCES project_statuses(id),
    changed_at  TIMESTAMPTZ DEFAULT NOW(),
    source_id   SMALLINT  REFERENCES data_sources(id),  -- 从哪个来源发现的这次变化
    remark      VARCHAR(500)
);

-- 数据源注册表
CREATE TABLE data_sources (
    id              SMALLINT PRIMARY KEY,
    name            VARCHAR(50)   NOT NULL,             -- 北极星风力发电网
    type            VARCHAR(20)   NOT NULL,             -- govt, bidding_platform, industry_media, wechat, corporate
    base_url        VARCHAR(200),
    scraper_module  VARCHAR(50),                        -- Python爬虫模块名
    is_active       BOOLEAN DEFAULT TRUE,
    scrape_interval_min INT DEFAULT 360,                -- 采集间隔(分钟)
    created_at      TIMESTAMPTZ DEFAULT NOW()
);

-- 采集运行日志
CREATE TABLE scrape_logs (
    id              BIGSERIAL PRIMARY KEY,
    data_source_id  SMALLINT     NOT NULL REFERENCES data_sources(id),
    started_at      TIMESTAMPTZ  NOT NULL,
    finished_at     TIMESTAMPTZ,
    items_scraped   INT DEFAULT 0,                      -- 本次抓取条数
    items_new       INT DEFAULT 0,                      -- 新增项目数
    items_updated   INT DEFAULT 0,                      -- 更新项目数
    status          VARCHAR(10)  DEFAULT 'running',     -- running, success, failed
    error_message   TEXT,
    created_at      TIMESTAMPTZ DEFAULT NOW()
);

-- ============================================
-- 索引
-- ============================================

-- 项目查询高频索引
CREATE INDEX idx_projects_province        ON projects(province_id);
CREATE INDEX idx_projects_type            ON projects(project_type_id);
CREATE INDEX idx_projects_status          ON projects(status_id);
CREATE INDEX idx_projects_tower_type      ON projects(tower_type_id);
CREATE INDEX idx_projects_approval_date   ON projects(approval_date DESC);
CREATE INDEX idx_projects_construction    ON projects(construction_start_date DESC);
CREATE INDEX idx_projects_completion      ON projects(completion_date DESC);
CREATE INDEX idx_projects_capacity        ON projects(capacity_mw);

-- 复合索引（覆盖高频组合查询）
CREATE INDEX idx_projects_type_status     ON projects(project_type_id, status_id);
CREATE INDEX idx_projects_prov_type       ON projects(province_id, project_type_id);
CREATE INDEX idx_projects_prov_status     ON projects(province_id, status_id);

-- 全文搜索（项目名称 + 业主 + 描述）
CREATE INDEX idx_projects_search           ON projects
    USING GIN (to_tsvector('simple', coalesce(name,'') || ' ' || coalesce(owner,'') || ' ' || coalesce(description,'')));

-- 来源关联索引
CREATE INDEX idx_project_sources_project   ON project_sources(project_id);
CREATE INDEX idx_project_sources_source    ON project_sources(source_id);

-- 日志索引
CREATE INDEX idx_status_logs_project       ON project_status_logs(project_id, changed_at DESC);
CREATE INDEX idx_scrape_logs_source        ON scrape_logs(data_source_id, started_at DESC);
```

## 二、API 接口设计

### 2.0 通用约定

**Base URL**: `https://api.example.com/v1`

**通用响应格式**:
```json
{
  "code": 0,
  "message": "success",
  "data": { ... },
  "pagination": {
    "page": 1,
    "page_size": 20,
    "total": 156,
    "total_pages": 8
  }
}
```

**错误码**:
| code | 含义 |
|------|------|
| 0    | 成功 |
| 400  | 请求参数错误 |
| 404  | 资源不存在 |
| 500  | 服务器内部错误 |

---

### 2.1 项目接口

#### `GET /projects` — 项目列表（支持多维筛选）

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | int | 否 | 页码，默认 1 |
| `page_size` | int | 否 | 每页条数，默认 20，最大 100 |
| `province_id` | int | 否 | 省份 ID |
| `region` | string | 否 | 大区（华东/西北…） |
| `project_type` | string | 否 | onshore / offshore |
| `status` | string | 否 | approved / bidding / construction / grid_connected / completed |
| `tower_type` | string | 否 | steel / hybrid / unknown |
| `capacity_min` | float | 否 | 容量下限(MW) |
| `capacity_max` | float | 否 | 容量上限(MW) |
| `approval_date_from` | date | 否 | 核准日期起 |
| `approval_date_to` | date | 否 | 核准日期止 |
| `completion_date_from` | date | 否 | 完工日期起 |
| `completion_date_to` | date | 否 | 完工日期止 |
| `owner` | string | 否 | 业主（模糊匹配） |
| `supplier` | string | 否 | 风机供应商（模糊匹配） |
| `keyword` | string | 否 | 全文搜索关键词 |
| `sort_by` | string | 否 | capacity_mw / approval_date / completion_date |
| `sort_order` | string | 否 | asc / desc，默认 desc |

### 3.2 LLM 提取 Prompt 设计

```
你是一个风电行业数据提取助手。从以下新闻/公告中提取风电项目的结构化信息。
如果某字段在原文中没有提及，请返回 null。

请返回严格的 JSON 格式，字段说明:
- project_name: 项目完整名称
- province: 所在省份

- unit_capacity_mw: 单机容量(兆瓦)，数字
- tower_type: "steel"(钢塔) / "hybrid"(混塔) / null(未提及)
- status: "approved" / "bidding" / "construction" / "grid_connected" / "completed"
- owner: 业主/开发商名称
- turbine_supplier: 风机供应商名称
- investment_bn: 投资金额(亿元)，数字
- approval_date: 核准日期 YYYY-MM-DD
- bid_date: 招标/中标日期 YYYY-MM-DD
- construction_start_date: 开工日期 YYYY-MM-DD
- completion_date: 完工/并网日期 YYYY-MM-DD
- planned_cod_date: 计划并网日期 YYYY-MM-DD

原文:
{article_text}
```
